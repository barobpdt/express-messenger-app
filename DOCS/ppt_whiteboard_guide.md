# 📊 PPT 화이트보드 사용 가이드

> **접속 주소:** `http://[서버IP]:8081/ppt_whiteboard.html`

## 1. 시작하기 — 방 설정

처음 접속하면 설정 모달이 표시됩니다.

| 항목 | 설명 |
|------|------|
| **역할** | `📢 발표자` 또는 `👁 참가자` 선택 |
| **방 코드** | 협업할 방의 코드 입력 (영문/숫자) |
| **닉네임** | 접속자 목록에 표시될 이름 |
| **서버 주소** | WebSocket 서버 주소 (기본값: 자동 감지) |

- **발표자**: 슬라이드 조작, 필기, 편집 등 모든 기능 사용 가능
- **참가자**: 발표자의 화면을 실시간으로 따라가며 관람

> 💡 같은 **방 코드**를 입력해야 같은 방에 접속됩니다.


## 2. 슬라이드 불러오기

### 방법 1: 파일 열기
헤더의 **📂 파일 열기** 버튼 → `.md` 또는 `.txt` 파일 선택

### 방법 2: 드래그 & 드롭
슬라이드 영역에 `.md` / `.txt` 파일을 드래그해서 놓기

### 마크다운 슬라이드 형식
슬라이드는 `---` 구분선으로 나뉩니다.

```markdown
# 첫 번째 슬라이드 제목

내용을 입력합니다.
- 항목 1
- 항목 2


# 두 번째 슬라이드

## 소제목

> 인용문도 사용 가능합니다.


# 세 번째 슬라이드

| 표 | 도 | 사용 가능 |
|----|----|----|
| A  | B  | C  |
```


## 3. 헤더 버튼 설명

| 버튼 | 기능 |
|------|------|
| **📂 파일 열기** | 마크다운 파일 불러오기 |
| **✏️ 편집** | 마크다운 에디터 열기 |
| **👥** | 접속자 사이드바 보기/숨기기 |
| **🔗** | 방 코드 클립보드에 복사 |
| **📥 PDF** | 전체 슬라이드를 PDF로 내보내기 |
| **🗑** | 현재 페이지 필기 전체 지우기 |
| **⚙** | 방 설정으로 돌아가기 (재접속) |


## 4. 하단 툴바 — 슬라이드 이동

```
◀  [현재 / 전체]  ▶
```

| 방법 | 동작 |
|------|------|
| `◀` / `▶` 버튼 클릭 | 이전/다음 슬라이드 |
| 키보드 `←` / `→` | 이전/다음 슬라이드 |
| 키보드 `↑` / `↓` | 이전/다음 슬라이드 |


## 5. 하단 툴바 — 필기 도구

| 도구 | 단축키 | 설명 |
|------|--------|------|
| ✏️ 펜 | `P` | 자유 곡선 그리기 |
| 📐 직선 | `L` | 직선 그리기 |
| ▭ 사각형 | `R` | 사각형 그리기 |
| ○ 원 | `C` | 원/타원 그리기 |
| 🔴 레이저 | `Z` | 레이저 포인터 (잔상 효과) |
| T 텍스트 | `T` | 클릭한 위치에 텍스트 입력 |
| 🖱 선택 | `V` / `Esc` | 필기 비활성화 (슬라이드만 보기) |

### 색상 선택
- 툴바의 색상 원 클릭으로 필기 색상 변경
- 마지막 🎨 버튼으로 커스텀 색상 선택

### 브러시 크기
- 툴바 슬라이더로 브러시 굵기 조절

### 페이지 필기 지우기
- 툴바 **현재 페이지 지우기** 버튼 또는 헤더 **🗑** 버튼


## 6. 마크다운 에디터

헤더의 **✏️ 편집** 버튼 또는 키보드 `E` 키

### 에디터 구성
- **왼쪽**: 마크다운 편집창 (모든 슬라이드를 하나의 문서로)
- **오른쪽**: 슬라이드별 실시간 미리보기

### 서식 툴바
| 버튼 | 기능 |
|------|------|
| `H1` `H2` `H3` | 제목 삽입 |
| **B** / *I* | 굵게 / 기울임 |
| `Code` | 인라인 코드 |
| `≡ Block` | 코드 블록 |
| `• 목록` | 순서 없는 목록 |
| `숫자` | 순서 있는 목록 |
| `❝ 인용` | 인용문 |
| `— 슬라이드 구분` | `---` 구분선 삽입 |
| `🖼️ 이미지` | 이미지 파일 업로드 |

### 이미지 삽입 방법
1. **버튼 클릭**: `🖼️ 이미지` → 파일 선택
2. **붙여넣기**: 이미지 복사 후 에디터에서 `Ctrl+V`
3. **드래그&드롭**: 이미지 파일을 편집창에 드롭

### 에디터 단축키
| 단축키 | 동작 |
|--------|------|
| `Tab` | 2칸 들여쓰기 |
| `Shift+Tab` | 2칸 내어쓰기 |
| `Enter` | 이전 줄 indent 유지 |
| `Enter` (목록 안) | 다음 목록 항목 자동 생성 |
| `Ctrl+Enter` | 변경 적용 & 에디터 닫기 |
| `Esc` | 에디터 닫기 (변경 취소) |


## 7. 실시간 협업

### 슬라이드 동기화
- **발표자**가 슬라이드를 이동하면 **참가자** 화면이 자동으로 따라감
- 에디터에서 적용(Ctrl+Enter)하면 모든 참가자에게 즉시 반영

### 필기 동기화
- 발표자가 그린 필기가 실시간으로 참가자 화면에 표시

### 레이저 포인터
- 발표자가 `Z` 키로 레이저 포인터 활성화 시 참가자 화면에도 같은 위치에 표시

### 접속자 확인
- 헤더 **👥** 버튼으로 현재 방에 접속한 사용자 목록 확인
- 각 사용자의 역할(발표자/참가자) 확인 가능


## 8. PDF 내보내기

헤더의 **📥 PDF** 버튼 클릭

- 모든 슬라이드를 순서대로 캡처
- **슬라이드 내용 + 필기(handwriting)** 모두 포함
- `presentation_YYYYMMDD.pdf` 파일로 자동 다운로드
- 진행 상황이 버튼에 표시됨 (예: `⏳ 3/8`)

> ⚠️ 슬라이드 수가 많을수록 시간이 걸립니다 (슬라이드당 약 0.5초)
