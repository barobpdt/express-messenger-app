## Verification Plan

### Manual Verification
1. 서버 실행: `node server.js` (이미 실행 중)
2. 브라우저 탭 2개에서 `http://localhost:8081/messenger.html` 접속 (서로 다른 계정 로그인)
3. 접속자 목록 → 상대방 옆 📞 버튼 클릭
4. 상대방 탭에 수신 팝업이 뜨는지 확인
5. 수락 → 두 탭 모두 "통화 중" 상태 바 표시 확인
6. 마이크 음소거 버튼 동작 확인
7. 통화 종료 → 양쪽 모두 idle 상태로 복귀 확인
