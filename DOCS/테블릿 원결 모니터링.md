# 태블릿 원격 모니터링 & 제어 시스템

기존 WebSocket 인프라를 활용해 태블릿의 JS 오류를 실시간으로 수집하고, **오류 발생 시 화면을 자동 캡처**하여 관리자 대시보드에서 실시간으로 확인하고 원격 제어(새로고침, JS 실행, 알림 팝업 등)할 수 있게 합니다.

---

## 아키텍처 흐름

```
[태블릿 order.html]                [서버 server.js]           [관리자 브라우저]
  tablet-agent.js                  WebSocket 허브             tablet-admin.html
      │                                  │                          │
      │── type:'tablet-init' ──────────>│                          │
      │── type:'tablet-error' ─────────>│── 포워딩 ──────────────>│
      │<─ type:'tablet-cmd' ───────────│<── 원격 명령 ────────────│
```

---

## 구현 항목

### 1. 태블릿 에이전트 스크립트
#### [NEW] [tablet-agent.js](file:///c:/bpdt/project/express-sample/public/js/tablet-agent.js)
- 기존 `order.js`의 `ws`를 공유하여 `tablet-init` 메시지로 자신을 등록
- 아래 이벤트를 캐치해 `type: 'tablet-error'`로 서버에 전송:
  - `window.onerror` (JS 런타임 오류)
  - `window.onunhandledrejection` (Promise 오류)
  - `fetch` 요청 실패 (네트워크/API 오류) — `fetch` 패치
- **오류 발생 시 `html2canvas`로 현재 화면 캡처 → base64 PNG → 에러 리포트에 포함**
- 관리자 요청(`tablet-cmd: 'screenshot'`)에도 즉시 캡처 후 전송
- `type: 'tablet-cmd'` 메시지 수신 시 아래 명령 처리:
  - `reload` : 페이지 강제 새로고침
  - `eval` : 전달된 JS 코드 실행
  - `alert` : 화면에 알림 팝업 표시
  - `navigate` : 지정 URL로 이동
  - `screenshot` : 즉시 화면 캡처 후 관리자로 전송

#### [MODIFY] [order.html](file:///c:/bpdt/project/express-sample/public/order.html)
- `html2canvas` CDN 스크립트 태그 추가
- `tablet-agent.js` 스크립트 태그 추가 (body 마지막)

### 2. 서버 WebSocket 메시지 처리
#### [MODIFY] [server.js](file:///c:/bpdt/project/express-sample/server.js)

`wss.on('message')` 핸들러에 아래 타입 추가:

| 수신 타입 | 처리 |
|---|---|
| `tablet-init` | `ws.isTablet = true`, `ws.tableId` 저장 |
| `tablet-error` | 관리자(`ws.isTabletAdmin`) 전체에 포워딩 |
| `tablet-admin-init` | `ws.isTabletAdmin = true` 등록 |
| `tablet-cmd` | `tableId`에 해당하는 태블릿 ws로 전달 |

---

### 3. 관리자 대시보드
#### [NEW] [tablet-admin.html](file:///c:/bpdt/project/express-sample/public/tablet-admin.html)

**기능:**
- 접속 중인 태블릿 목록 (테이블 번호, 마지막 활동)
- 실시간 오류 피드 (태블릿 ID, 오류 메시지, 파일/라인 번호, 시각)
- **오류 카드에 화면 캡처 썸네일 표시 → 클릭 시 원본 크기로 확대**
- 원격 제어 패널:
  - 🔄 **새로고침**: `{ type:'tablet-cmd', cmd:'reload' }`
  - 📸 **화면 캡처**: `{ type:'tablet-cmd', cmd:'screenshot' }` (즉시 스크린샷 요청)
  - 💻 **JS 실행**: 코드 입력 후 `{ type:'tablet-cmd', cmd:'eval', code:'...' }`
  - 📢 **알림 팝업**: `{ type:'tablet-cmd', cmd:'alert', message:'...' }`
  - 모든 태블릿 or 특정 테이블 선택 전송

## 보안 고려사항

> [!IMPORTANT]
> `eval` 명령은 관리자 대시보드에서만 사용 가능하도록 `ws.isTabletAdmin` 플래그로 접근을 서버에서 제한합니다.
> 실제 운영 시 관리자 인증(JWT 등) 추가를 권장합니다.

## 검증 계획

1. 태블릿(`order.html`)에서 의도적 오류 발생 → 관리자 대시보드에 오류 표시 확인
2. 관리자 대시보드에서 "새로고침" 명령 → 태블릿 페이지 새로고침 확인
3. 관리자 대시보드에서 JS 코드 실행 → 태블릿 화면 반영 확인
