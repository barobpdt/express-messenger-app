# Funny Music UI 프리미엄 리디자인 계획

"Funny Music" 인터페이스를 구식 네온 스타일에서 현대적이고 고급스러운 "사이버 프리미엄" 디자인으로 개편합니다.

## 제안된 변경 사항

### [프런트엔드]

#### [수정] [funny_music.html](file:///c:/bpdt/project/express-sample/public/funny_music.html)
- **루트 변수 리팩토링**: 더욱 세련된 색상 팔레트(예: 딥 인디고, 소프트 시안, 바이브런트 바이올렛)와 더 나은 간격 토큰을 도입합니다.

- **Glassmorphism 통합**: 사이드바, 노래 목록 및 모달에 `backdrop-filter: blur()`를 적용하여 반투명 배경을 만듭니다.

- **헤더 개편**: 내비게이션을 정리하고 로고 서체를 개선하며 상단 바에 미묘한 효과를 추가합니다.

- **사이드바(메뉴) 재디자인**: 더 나은 패딩, 둥근 모서리, 부드러운 호버 효과를 적용한 최신 아이콘 버튼 레이아웃을 사용합니다.

- **중앙 영역(무대 및 비디오)**:

- 날카로운 청록색 테두리를 부드럽고 빛나는 그라데이션으로 교체합니다.

- CSS 방사형 원뿔 그라데이션을 사용하여 "무대 조명" 효과를 개선하여 더욱 사실적인 느낌을 줍니다.

- 가사 표시를 개선된 서체와 가독성으로 업데이트합니다.

- **오른쪽 패널(곡 목록)**:

- 곡 목록의 밀도와 가독성을 개선합니다.

- 행 강조 표시와 "현재 재생" 표시를 추가합니다.

- **모달 및 알림**:

- 예약 및 알림 모달을 완전히 재디자인하고 고급스러운 모서리 처리와 그림자를 적용합니다.

- 토스트 알림 시스템을 개선하여 가시성과 현대적인 디자인을 향상시킵니다.

## 검증 계획

### 자동화 테스트
- 해당 없음 (UI 중심)

### 수동 검증
1. 사용자의 브라우저에서 `funny_music.html`을 엽니다.

2. "사이버 프리미엄" 디자인(글래스모피즘, 은은한 빛)이 제대로 구현되었는지 확인합니다.

3. 모든 버튼의 상호 작용 상태(호버, 활성)를 확인합니다.

4. "예약 신청" 모달을 열고 새로운 디자인이 제대로 표시되는지 확인합니다.

5. "직원 호출" 버튼과 해당 버튼의 펄스 애니메이션을 테스트합니다.

6. 노래 목록이 여전히 읽기 쉽고 제대로 작동하는지 확인합니다.

7. 기존 JavaScript 로직(전화, 모달)이 모두 정상적으로 작동하는지 확인합니다.
