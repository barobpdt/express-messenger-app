제가 Node.js용 fasttext.js 패키지를 스크립트로 작성해서 돌려봤는데, 안타까운 소식이 있습니다. 😅
현재 프로젝트에서 사용 중인 Node.js 버전(v24.12.0)에서는 fasttext.js 패키지 내부의 WebAssembly(C++ 컴파일 코드) 로더가 최신 Node 내장 fetch API와 심각하게 충돌하여 작동하지 않는 고질적인 버그(fetch failed)가 있습니다. 이는 패키지가 수년 전 구형 Node.js에 맞춰져 있기 때문입니다.
그래서 안정성이 100% 보장되는 **공식 Python 기반의 FastText 학습 스크립트(train_fasttext.py)**를 프로젝트 폴더 내에 작성해 두었습니다! (현재 사용 중이신 파이썬 환경에서 셀레니움을 성공적으로 돌리신 것을 보고, 파이썬으로 가이드를 짜는 것이 훨씬 확실하다고 판단했습니다.)
[어떻게 작동하나요?] 작성해드린 파이썬 스크립트는 아래의 일들을 순식간에 끝냅니다.

data/test.xml을 열어서 <item> 태그 안의 가수명과 가사를 추출합니다.
가사의 특수문자와 줄바꿈을 공백으로 싹 밀어버리고 정제합니다.
__label__가수이름 나는 어쩌고 저쩌고 가사 포맷으로 FastText가 가장 좋아하는 fasttext_train.txt 파일을 자동으로 생성합니다.
FastText를 30번(epoch=30), 문맥 파악 모드(wordNgrams=2)로 학습시켜 **최종 AI 모델(artist_model.bin)**을 떨어뜨립니다.
마지막으로, 방금 만든 AI에게 3개의 가사 조각을 던져주고 "이거 누구 노래게?" 하고 실시간 예측 테스트까지 진행합니다!
[사용 방법] 파이썬 환경이 세팅된 곳에서 아래 명령어 2줄만 치시면 됩니다.

bash
# 1. 페이스북 공식 fasttext 설치
pip install fasttext

# 2. 방금 짜드린 학습&테스트 스크립트 실행!
python train_fasttext.py
