Core Features

클립보드 하나로 이 모든 것이 가능합니다

복사한 순간부터 관리·공유·발표까지 끊김 없는 워크플로우를 경험하세요.

📂

파일 탐색기 통합 관리

탐색기에서 복사한 파일 목록을 즉시 확인하고, 이동·복사·삭제·압축·미리보기 등 핵심 작업을 한 화면에서 처리합니다. 드래그 앤 드롭으로 직관적인 파일 조작이 가능합니다.

파일 이동 복사 / 삭제 압축 미리보기
🔍

파일 형식별 상세 뷰

이미지, 압축 파일, 소스 코드, 미디어 파일 등 확장자에 따라 최적화된 미리보기와 인라인 플레이 기능을 제공합니다.

이미지 ZIP / RAR MP4 / MP3 소스코드
📦

압축 풀기 & 백업

ZIP, RAR, 7Z 등 주요 압축 포맷을 한 번에 풀고, 중요 파일을 자동 백업 폴더로 아카이빙합니다.

압축 해제 자동 백업 아카이브
🌐

웹 스크랩 자동화

웹 페이지에서 복사한 텍스트·이미지·링크가 자동으로 스크랩되어 원본 출처와 함께 클립보드 히스토리에 저장됩니다.

자동 저장 출처 기록 태그 분류
📸

이미지 캡쳐 → 메신저 공유

화면 캡쳐 즉시 팀 메신저(웹뷰 메신저 포함)로 바로 공유합니다. 캡쳐·첨부·전송이 단축키 하나로 완성됩니다.

영역 캡쳐 즉시 공유 메신저 연동
🎬

문서 병합 & 실시간 전자 화이트보드

복사한 여러 문서를 하나로 병합하고, 실시간 프리젠테이션 모드로 발표하거나 전자 화이트보드로 협업합니다. PPT·Word·PDF를 별도 프로그램 없이 즉시 슬라이드로 변환합니다.

문서 병합 슬라이드 변환 실시간 발표 화이트보드
⚖️

파일 비교 & 저장소 관리

두 파일을 나란히 비교(diff)하고, 클립보드 이력을 버전별로 저장소에 관리합니다. 언제든 과거 복사 항목으로 복원할 수 있습니다.

How It Works

3단계로 시작하세요

⌨️

1. 그냥 복사하세요

탐색기, 브라우저, 어디서든 Ctrl+C 를 누르면 Clipboard Magic이 자동으로 감지하여 히스토리에 저장합니다.

✨

2. 마법처럼 정리됩니다

파일 형식을 자동 분류하고, 이미지는 미리보기로, 영상은 플레이어로, 코드는 하이라이터로 최적 표시합니다.

🚀

3. 원하는 작업을 하세요

이동, 공유, 발표, 비교 — 필요한 기능을 바로 실행합니다. 메신저 공유 버튼 하나로 팀원에게 즉시 전달됩니다.
