퍼니뮤직 시스템 대시보드

이 대시보드는 퍼니뮤직 앱 및 시스템 관리 유틸리티의 중앙 허브입니다.
좌측의 메뉴바를 통해 각 애플리케이션에 접근할 수 있습니다. 아래에서 각 모듈의 명칭과 상세 역할을 확인하세요.

음악 및 노래방 시스템

노래방 메인 플레이어
funny_music.html
유튜브 웹 기반의 노래방 메인 화면입니다. 영상 재생 제어, 예약 목록 관리, 오디오 피치 다운(키 조절), 간주 점프 등의 핵심 기능을 담당합니다.
노래 검색 리모컨
music_select.html
노래방 번호책을 대신하는 검색형 리모컨 화면입니다. 초성 검색, 가수/제목 검색이 가능하며, 선택한 곡을 메인 플레이어에 예약 전송합니다.
실시간 가사 뷰어
lyrics.html
가사를 별도의 디스플레이나 화면에 크게 띄워주는 보조 뷰어입니다. Server-Sent Events 및 WebSocket을 통해 메인 화면과 재생 상태를 동기화하여 실시간으로 가사를 추적합니다.

업무 및 스케줄 관리

영상재생 스케줄 관리
schedule-manager.html
FullCalendar를 활용하여 시간대 및 요일별로 동영상 재생 일정을 캘린더 형태에서 시각적으로 관리합니다. 영상을 직접 파일 업로드하거나 예약할 수 있습니다.
스케줄형 영상 플레이어
video-player.html
schedule-manager.html 에서 등록된 현재 시간대의 일정에 맞춰 서버로부터 동영상을 스트리밍 받아 자동으로 끊김 없이 연속 재생하는 Kiosk용 무인 플레이어입니다.
테블릿 음식 주문
order-management.html
테이블 비치용 음식/주류 주문 시스템입니다. 장바구니에 아이템을 담고 총액을 확인하며 주방으로 주문을 접수하는 모던한 UI를 제공합니다.

데이터베이스 클라이언트

PostgreSQL 클라이언트
pg-client.html
웹 기반의 DB 관리 툴입니다. 연결된 원격 DB의 스키마 다이어그램 구조를 트리 형태로 탐색할 수 있으며 Monaco 코딩 에디터를 이용해 직접 SQL 질의 및 차트 시각화를 수행합니다.
SQLite 클라이언트
sqlite-client.html
로컬 서버의 SQLite DB 파일(.sqlite) 경로를 입력하여 직접 연결하고 자유롭게 쿼리를 조회하며 테이블 구조를 분석할 수 있는 웹 스튜디오 환경입니다.

통신 및 파일 시스템

웹 브라우저 메신저
messenger.html
Socket.IO 기반 실시간 오픈 채팅 및 DM 전송 프로그램입니다. 방(Room) 생성, 접속자 목록 표시 등을 지원합니다.
WebRTC 데이터 통신
webrtc.html
WebRTC Data Channel을 이용해 서버 부하 및 우회 없이 클라이언트 간 브라우저 1:1 직접 통신(P2P) 상태를 체크하고 텍스트를 교환하는 테스트 환경입니다.
파일 서버 폴더 탐색기
file-share.html
서버 시스템 내부의 실제 로컬 디스크 구조(C드라이브, D드라이브 등)를 원격으로 탐색하고 파일을 다운로드하거나 새 폴더를 생성하고 패키지를 업로드할 수 있는 웹 탐색기 시스템입니다.
P2P 파일 전송기
file-transfer.html
WebRTC 기반의 클라이언트 대 클라이언트 P2P 대용량 파일 전송 서비스입니다. 서버 스토리지를 거치지 않고 상대측 브라우저로 양방향 직통 청크(Chunk) 단위 전송을 수행합니다.
