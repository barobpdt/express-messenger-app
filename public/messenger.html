💡 명령어 도움말

  • /avatar [URL]: 프로필 사진 변경
  • /open [URL]: 상대방 브라우저에 웹페이지 열기
  • /alert [메시지]: 상대방 화면에 알림창 띄우기
  • /screenshot: 상대방 모니터 화면 캡처 요청
  • /sharescreen: 화면 공유 시작
  • /login: 로그인 / 계정 전환
  • /cmd [명령어]: 서버 터미널에서 명령어 실행
  • /log: 서버 로그 보기
  • /monitor [경로]: 파일 변경사항 실시간 감시
  • /unmonitor: 파일 감시 중지
